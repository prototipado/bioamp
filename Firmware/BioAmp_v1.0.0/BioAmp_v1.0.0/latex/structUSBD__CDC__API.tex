This function is called by application layer to send C\+DC class notifications to host. See usbcdc11.\+pdf, section 6.\+3, Table 67 for various notification types the C\+DC device can send. \begin{DoxyNote}{Note}

 For all other notifications application should construct the notification buffer appropriately and call hw-\/$>$U\+S\+B\+\_\+\+Write\+E\+P() for interrupt endpoint associated with the interface.
